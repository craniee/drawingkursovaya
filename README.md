# Generationpicture

Программа генерации случайного рисунка.

## Функционал
- выводит окно с интерфейсом
- можно настроить кучность
- настроить размерную сетку
- выбрать тип фигур
- написать количество фигур


## Технологии
- Java 17.0.17
- Gradle 9.0.0
- JavaFX 21
- Apache Log4j2

```bash
.\run.bat
